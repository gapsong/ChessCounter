# ChessCounter
Helps you to improve in chess


```shell
$ yarn install
```

```shell
$ yarn dev
```
1. Open lichess.org
2. Open console from browser
3. Paste generated ```main.js``` file

Copy generated main.js file script to the browser

It works just for **white** at the moment
